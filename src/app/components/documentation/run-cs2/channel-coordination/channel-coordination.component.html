Channel Coordination

SciFlow uses channels to facilitate communication between modules. When one end of the channel receives a string, this signals the completion of a module, leading to the invocation of the next module. The string received, is now sent to the next step.

The general code structure for channel coordination is explained in Step 04 : Writing your control thread.
