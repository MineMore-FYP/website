Running the framework (for the hello world app)

Now that you have successfully installed the required software, we can move on to using SciFlow to build your own workflow!

  1. Download the structure of the repository from the following GitHub link: https://github.com/SciFlow-FYP/SciFlow
  2. Open up your Linux terminal.
  3. Navigate to the SciFlow framework you just downloaded.
  4. Run the following command. 
    go run controlThread.go

If you see the following output on your terminal, the framework is ready for your changes!
