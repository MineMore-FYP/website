Pre-requisites

In order to easily grasp concepts and unlock the full potential of the SciFlow framework, basic knowledge of programming with python and golang is necessary.

SciFlow has been tested on the Ubuntu 16.04 LTS Operating System.

We are working on making SciFlow available on other operating systems as well. We will keep you posted!

Before you begin, make sure your development environment includes the following;

  • Python 3.5.2
  • Pandas 0.17.1
  • Numpy 1.17.4
  • Scikit-learn 0.21.3
  • Matplotlib 1.5.1
  • Parsl 0.9.0
  • Golang 1.13.5
  • Any text editor or IDE (eg: Vim, gedit, Atom, Visual Studio Code)

You can easily install the above by following the installation instructions given below.

Python 3.5.2 https://realpython.com/installing-python/
Pandas 0.17.1 pip3 install pandas==0.17.1
Numpy 1.17.4 pip3 install numpy==1.17.4
Scikit-learn 0.21.3 pip3 install scikit-learn==0.21.3
Matplotlib 1.5.1 pip3 install matplotlib==1.5.1
Parsl 0.9.0 pip3 install parsl==0.9.0
Golang 1.13.5 https://golang.org/dl/
Vim sudo apt install vim

Running the framework (for the hello world app)

Now that you have successfully installed the required software, we can move on to using SciFlow to build your own workflow!

  1. Download the structure of the repository from the following GitHub link: https://github.com/SciFlow-FYP/SciFlow
  2. Open up your Linux terminal.
  3. Navigate to the SciFlow framework you just downloaded.
  4. Run the following command. 
        
  go run controlThread.go

If you see the following output on your terminal, the framework is ready for your changes!
